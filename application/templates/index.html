this project is deployed from Virtual box Linux Instance

-create heroku account
-then download herokucli,git bash
-use cmd to access heroku
-heroku login
-python3 -m django startproject project
-cd to project
-python3 manage.py startapp application
-cd to application
-go to views.py and add index()
-go back and cd to project
-update urls.py
-update settings.py
-go to project home directory
-create Procfile
//Procfile:
web: gunicorn project.wsgi --log-file -
// end of Procfile
-create runtime.txt : python 3.7.9
-create requirements.txt:
-django
-gunicorn
-django-heroku
-git init
-git login
-git add .
-git commit -m "first"
-heroku create
-heroku config:set DISABLE_COLLECTSTATIC=1
-git push heroku main
-heroku open
